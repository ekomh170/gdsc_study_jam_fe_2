### Identifikasi Kebutuhan Untuk Membuat Halaman Kartu Profil (Card Profile)

### Struktur Proses Berfikir Sebagai Frontend (FE) :
- Kotak Kartu Profile 
    - Biodata Profil - Pengguna Aplikasi
        - Gambar Profil 
        - Nama Profil
        - Umur Profil
        - Lokasi Profil
    - Status Profile - Pengguna Aplikasi
        - Jumlah Followers 
        - Jumlah Suka 
        - Jumlah Foto
